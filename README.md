# Osmize.com modern mock-up

Mock-up non ufficiale per proporre un redesign di Osmize.com.

L'obiettivo e rendere il portale piu semplice, moderno e fruibile dal pubblico:

- apertura immediata su "osmize aperte oggi";
- schede leggibili con localita, date, telefono e percorso;
- ricerca per nome o localita;
- filtro per zona;
- struttura pronta per GitHub Pages;
- messaggio chiaro per proporre una collaborazione ai responsabili del sito.

## Come vederlo

Apri `index.html` nel browser oppure pubblica la cartella con GitHub Pages.

## Nota sui dati

I contenuti sono dimostrativi e ricavati dalla struttura pubblica consultabile su Osmize.com il 30/06/2026. Prima di un uso reale servira collegare il progetto a una fonte dati aggiornata e autorizzata.
